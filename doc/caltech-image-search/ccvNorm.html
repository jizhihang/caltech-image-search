ccvNorm

PURPOSE

CCVNORM returns the norm of the input data

SYNOPSIS

function norm = ccvNorm(data, type, mx)

DESCRIPTION 

 CCVNORM returns the norm of the input data
 
 INPUTS
 ------
 data    - the input data, with one column per example
 type    - ['l2'] the normalization type
           'l1'  - L1 normalization
           'l2'  - L2 normalization
           
 
 OUTPUTS

 ndata   - the normalized data
